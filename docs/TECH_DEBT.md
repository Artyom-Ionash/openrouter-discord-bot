## 1. Модульная система инструментов (Регистрация)

Чтобы свободно переключаться между API (OpenRouter, Gemini и др.):

1.  **Интерфейс:** Определите `core/integrations/base_llm.py` (абстрактный базовый класс или протокол `Protocol`).
2.  **Реализация:** Создайте `OpenRouterProvider`, `GeminiProvider`.

## 🟢 Низкий приоритет

1.  **Debugger Race Condition (VS Code):** Исправить проблему «тихого вылета» (silent exit) при первом запуске через отладчик VS Code.
    - **Суть:** Процесс завершается с кодом 0 без ошибок из-за конфликта инициализации `debugpy` и асинхронного цикла событий в момент установки соединения с Discord Gateway.
    - **Временное решение:** Повторный запуск или запуск без отладчика через терминал.
